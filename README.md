# Work flow app

A web app to manage tasks

Currently hosted at

- [workflowboard.netlify.app](https://workflowboard.netlify.app/)

## Packages used

idb, Tailwind CSS

## How to use the app

App defaults to board view.

1. Working with tasks

Create task:
Click "New task"
Fill details
Click "Save"

Edit task:
Click any task to open it
Click "Edit"
Edit the details
Click "Update"

Delete task
Click any task to open it
Click "Delete"
Click "Confirm"

2. Search and Filter

Search:
In the search box type the text
Matching tasks will be displayed

Filter:
Click "Filter"
Select "Available filter option"
Matching task will be displayed
